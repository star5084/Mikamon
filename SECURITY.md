# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Mikamon, please report it responsibly:

* Email: star.5084.mikamon@gmail.com
* Provide clear steps to reproduce the issue
* Include affected versions if known

We aim to respond within 7 days.

## Supported Versions

This policy applies to the latest released version of Mikamon.

## Disclosure Timeline

We follow a 90-day disclosure timeline from the initial report to public disclosure, unless otherwise agreed.

Thank you for helping keep Mikamon safe!
